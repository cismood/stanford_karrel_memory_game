# Croatia footbal Memory Game 

Memory game developed in Python using lib Pygame.

## Features

- Croatian national team theme
- Real card flip animation
- Timer
- Move counter
- Score counter
- Last-click highlight
- Restart game with R key
- Win screen

## Requirements

Python 3.11

## Installation

pip install -r requirements.txt

## Run

python main.py


## IMPORTANT ##

If you have problem runing the game, just hit this comands in bash and it will run in virtual enviroment. 

python3.11 -m venv test311
source test311/bin/activate

python -m pip install --upgrade pip
pip install -r requirements.txt

python main.py

Happy playing!
